# GreenCare Lawn Services

## Overview

GreenCare Lawn Services is a high-fidelity React prototype developed for **SEG 3125 – Analysis and Design of User Interfaces** at the University of Ottawa.

The project demonstrates the design and implementation of a lawn care service website that allows customers to request services online and allows service providers to manage incoming bookings through a simple administrative interface.

The prototype was developed following a user-centered design process using personas, storyboards, visual design principles, and iterative improvements.

## Design Decisions

### Visual Design

Customer-facing screens use:

* Green color palette
* Lawn imagery
* Rounded buttons
* Friendly visual style

Administrative screens use:

* Blue and grey color palette
* Dashboard layout
* Status indicators
* Professional appearance

### UX Principles Applied

* Clear visual hierarchy
* Consistent navigation
* Strong call-to-action buttons
* Minimal information collection
* Immediate user feedback
* Mobile-first design approach
